The Solution: Environmental Hazard Alert System

My final project is a compact, dedicated safety device built on the **Micro:bit platform** designed to monitor two critical, often unseen, environmental hazards: combustible gases and airborne dust particles. This system leverages the Micro:bit shield for stable connections and Python for logic processing.

Core Hardware Components

  • **Micro:bit & Micro:bit Shield (Elecfreaks):** The central controller and stable connection board.
  • **MQ-2 Gas Sensor:** Detects hazardous gases like Butane, LPG, and Methane.
  • **Dust Particle Sensor:** Monitors the concentration and size of airborne dust.
  • **Passive Buzzer:** Provides immediate, audible alerts for unsafe levels.
  • **OLED Display (Optional/Initial Test):** Used for providing digital readings during testing and final display.

Functionality and Logic

The device is programmed in **Python (MicroPython)**. It continuously reads analog values from the sensors and compares them against pre-set safety thresholds.

  • **Gas Threshold:** Alert triggers when the MQ-2 sensor reading reaches or exceeds **400 parts per million (PPM)**.
  • **Dust Threshold:** Alert triggers when the dust sensor detects particle concentrations indicating an **unsafe level** (e.g., in a heavy woodworking environment).
  • **Alarm System:** The passive buzzer is activated immediately when *either* the gas or dust threshold is met, providing a crucial safety warning.

// Conceptual Python Logic Snippet (Illustrating Analog Reading and Comparison)

from microbit import *
import music

GAS_THRESHOLD = 400 

def read_gas():
    # Assuming MQ-2 is connected to analog pin P0
    return pin0.read_analog() 

def check_hazards():
    gas_level = read_gas()
    # Assume dust_level check is also included here
    
    if gas_level >= GAS_THRESHOLD: 
        # Trigger alarm if gas is too high
        music.play(music.BA_DING, wait=False)
        display.show(Image.SAD)
    else:
        # Keep device quiet if safe
        display.show(Image.HEART)
        music.stop()

while True:
    check_hazards()
    sleep(500)

Real-World Impact

The system is designed to provide safety in two key scenarios:

  • **Gas Leaks:** Detecting rising methane/LPG levels from an unattended stove, alerting the user before dangerous concentrations are reached.
  • **Air Quality:** Providing a warning in environments like home workshops where high levels of fine dust particles can pose serious respiratory risks.
